# PRD: little-garden 学习模块

**项目**: little-garden
**提案ID**: P-20260513-001
**版本**: v1.0
**日期**: 2026-05-13
**状态**: approved_for_dev
**摘要**: 实现 GDD 中设计的4大学习场景（数学/语文/英语/逻辑），构建游戏核心循环

---

## 1. 背景与目标

### 1.1 问题陈述
当前 little-garden 仅有花园种植基础功能，GDD 设计的4大学习模块（数学/语文/英语/逻辑）未实现。游戏副标题"构建你的牌组，击败所有敌人！"是空头承诺。

### 1.2 目标
- 实现完整的学习关卡系统
- 每学科3个难度级别，每级3个关卡 = 共36个关卡
- 关卡完成后解锁奖励选择（类似 DBG 牌组构建）
- 构建"学习 → 奖励 → 强化 → 继续学习"的核心循环

---

## 2. 功能需求

### 2.1 学习场景入口

**数学花园**
- 场景配色：#FFF9C4 淡黄背景 + #FFE082 暖黄地面
- NPC：小狐狸（聪明、数数专家）
- 关卡类型：数数、加减法、形状识别

**语文花园**
- 场景配色：#F3E5F5 淡紫背景 + #E1BEE7 紫灰地面
- NPC：小猫（好奇、汉字专家）
- 关卡类型：汉字学习、拼音认知

**英语花园**
- 场景配色：#E3F2FD 淡蓝背景 + #BBDEFB 天空蓝地面
- NPC：小鸟（爱唱歌、字母专家）
- 关卡类型：字母认知、单词配对

**逻辑花园**
- 场景配色：#E0F7FA 淡青背景 + #B2EBF2 青白地面
- NPC：小鹿（优雅、逻辑专家）
- 关卡类型：分类、排序、规律发现

### 2.2 关卡结构

```
LEARNING = 'learning'

每关卡包含：
- 题目展示区（Canvas 中部）
- 答案选项区（底部，2-4个选项）
- 星星计数（3星制）
- 返回按钮（HUD）
- 反馈动画（正确/错误）
```

### 2.3 关卡类型详细设计

#### 数学关卡

| 类型 | 描述 | 难度递进 |
|------|------|----------|
| 数数 | 数苹果/花朵数量，选择正确答案 | L1:1-5, L2:1-10, L3:1-20 |
| 加法 | 两堆物品相加 | L1:1+1~5, L2:1+1~10, L3:多位数 |
| 减法 | 拿走物品后剩几个 | L1:1-1~3, L2:1-1~5, L3:多位数 |
| 形状识别 | 选择正确的形状 | L1:圆形/方形, L2:+三角形, L3:+多边形 |

#### 语文关卡

| 类型 | 描述 | 难度递进 |
|------|------|----------|
| 汉字认知 | 看图选汉字 | L1:5个简单字, L2:10个, L3:20个 |
| 拼音认知 | 选择正确拼音 | L1:单韵母, L2:声母, L3:完整音节 |
| 看图说话 | 选择正确汉字 | L1:名词, L2:动词, L3:形容词 |

#### 英语关卡

| 类型 | 描述 | 难度递进 |
|------|------|----------|
| 字母认知 | 选择正确字母 | L1:A-E, L2:A-J, L3:A-Z |
| 单词配对 | 图片与单词配对 | L1:5词, L2:10词, L3:20词 |
| 字母排序 | 按顺序排列字母 | L1:3字母, L2:5字母, L3:随机段 |

#### 逻辑关卡

| 类型 | 描述 | 难度递进 |
|------|------|----------|
| 分类 | 将物品归类到正确类别 | L1:2类, L2:3类, L3:多属性 |
| 排序 | 按大小/颜色/数量排序 | L1:3项, L2:5项, L3:7项 |
| 找规律 | 推断下一个图案 | L1:ABAB, L2:ABCABC, L3:复杂 |

### 2.4 奖励系统

**关卡结算**
- 3星：全对 → 3个奖励选项
- 2星：1次错误 → 2个奖励选项
- 1星：2次错误 → 1个奖励选项

**奖励类型**
- 新角色碎片（小园宝/小蜜蜂升级）
- 花园装饰（解锁新植物）
- 特殊道具（双倍积分卡、跳过卡）

### 3.2 渲染结构

```

- HUD层（星星 + 返回）
```
