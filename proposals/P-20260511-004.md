## ChatDev 概述

ChatDev 是从专用软件开发多 Agent 系统演进而来的**综合性多 Agent 编排平台**。

### 两个版本

1. **ChatDev 2.0 (DevAll)**：零代码多 Agent 编排平台，用户可通过简单配置快速构建和执行定制化多 Agent 系统
2. **ChatDev 1.0 (Legacy)**：虚拟软件公司模式，通过各种智能 Agent（CEO、CTO、Programmer 等）参与专业功能研讨会

### 核心特性

- **零代码平台**：无需编码，通过配置定义 Agent、工作流和任务
- **多 Agent 协作**：支持数据可视化、3D 生成、深度研究等复杂场景编排
- **Puppeteer 范式**：利用可学习的中央编排器，通过强化学习优化，动态激活和排序 Agent
- **MacNet**：多 Agent 协作网络，利用有向无环图促进 Agent 间有效协作

### 技术演进

- **ChatDev 1.0**：链式拓扑结构
- **MacNet**：支持多种拓扑结构，支持超过 1000 个 Agent
- **Puppeteer**：中央编排器 + 强化学习
- **IER (Iterative Experience Refinement)**：讲师和助理 Agent 增强快捷经验

## 技术架构

### 核心模块

1. **Agent 系统**：CEO、CTO、Programmer、Reviewer、Designer 等角色
2. **Chain 系统**：ChatChain 配置，工作流编排
3. **可视化系统**：实时查看 Agent 协作过程
4. **经验学习**：Experiential Co-Learning、IER

### 模式支持

- **Art Mode**：设计 Agent 生成图像
- **Human-Agent-Interaction**：人类参与评审
- **Git Mode**：版本控制
- **Docker Mode**：安全执行环境
- **Incremental Development**：增量开发
