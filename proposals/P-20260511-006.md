## 项目目标

基于 deepseek-ai/DeepSeek-Coder 开源代码生成模型，创建设计文档项目，对代码语言模型进行系统性分析。

## DeepSeek-Coder 概述

DeepSeek-Coder 是由 DeepSeek 团队开发的**代码语言模型系列**，从零开始在 2T tokens 上训练，其中 87% 为代码，13% 为中英文自然语言。

### 核心特性

- **海量训练数据**：2T tokens，87% 代码 + 13% 自然语言
- **多模型规模**：1B、5.7B、6.7B、33B 四种规模
- **超长上下文**：16K 窗口大小，支持项目级代码补全
- **Fill-in-the-Blank**：额外填空任务，支持代码填充
- **多语言支持**：支持 100+ 编程语言

### 性能表现

- HumanEval Python、MBPP、DS-1000、APPS 等基准测试 state-of-the-art
- DeepSeek-Coder-Base-33B 超越所有开源代码模型
- DeepSeek-Coder-Base-7B 达到 CodeLlama-34B 性能
- DeepSeek-Coder-Instruct-33B 超越 GPT-3.5 Turbo

## 技术架构

### 数据处理流程

1. **数据收集**：从 GitHub 收集代码，应用 StarCoder 过滤规则
2. **依赖解析**：解析同一仓库内文件的依赖关系，重排文件位置
3. **文件拼接**：将依赖文件拼接为单一示例，进行 repo-level minhash 去重
4. **质量过滤**：过滤语法错误或可读性差的低质量代码

### 模型训练流程

1. **预训练 I**：87% 代码 + 10% 代码相关语言 + 3% 非代码相关中文，1.8T tokens，4K 窗口
2. **预训练 II**：扩展 16K 窗口，额外 200B tokens，形成 DeepSeek-Coder-Base
3. **指令微调**：2B tokens 指令数据，形成 DeepSeek-Coder-Instruct

### 模型版本

- **DeepSeek-Coder-Base**：基础预训练模型
- **DeepSeek-Coder-Instruct**：指令微调模型
