## 项目目标

基于 HKUDS/DeepCode 开源项目，创建设计文档项目，对多 Agent 代码生成系统进行系统性分析。

## DeepCode 概述

DeepCode 是一个**开放 Agentic 编程框架**，通过多 Agent 系统推进代码生成。

### 核心特性

- **多 Agent 系统**：利用多个 Agent 协作完成复杂代码生成任务
- **开放 Agentic 编程**：支持开放式代码生成和修复
- **代码智能分析**：代码补全、漏洞检测、代码优化
- **多语言支持**：Python 3.13+

### 技术基础

- **论文**：[arXiv:2512.07921](https://arxiv.org/abs/2512.07921)
- **研究团队**：HKU Data Intelligence Lab
- **社区支持**：Discord、微信群

## 技术架构

### 核心模块

1. **多 Agent 协作系统**：多个 Agent 分工协作
2. **代码生成引擎**：基于 LLM 的代码生成
3. **代码分析模块**：代码补全、漏洞检测
4. **质量评估**：生成代码的质量评估

### 技术栈

- Python 3.13+
- LLM（支持多种模型）
- Multi-Agent Systems
