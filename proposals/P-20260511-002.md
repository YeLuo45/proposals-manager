## AutoAgent 概述

AutoAgent 是一个**全自动零代码**的 LLM Agent 框架，用户可以通过自然语言创建和部署 Agent 系统。

### 核心特性

- **自然语言驱动的 Agent 构建**：通过对话自动构建和编排协作式 Agent 系统
- **零代码框架**：无需编程经验即可创建定制化的 Agent、工具和工作流
- **自管理的工作流生成**：根据高级任务描述动态创建、优化和调整 Agent 工作流
- **智能资源编排**：通过迭代自改进生成受控代码来创建工具、Agent 和工作流
- **自博弈 Agent 定制**：支持单 Agent 创建和多 Agent 工作流生成

### 三大模式

1. **User Mode（深度研究 Agent）**：开箱即用的多 Agent 系统，作为综合 AI 研究助手
2. **Agent Editor**：通过自然语言创建独立的 Agent（不绑定工作流）
3. **Workflow Editor**：通过自然语言创建带工作流的 Agent

## 技术架构

### 目录结构

```
AutoAgent/
├── autoagent/              # 核心包
├── constant.py             # 常量定义
├── process_tool_docs.py     # 工具文档处理
├── tool_docs.csv           # 工具文档
├── loop_utils/             # 循环工具
├── process_tool_docs.py    # 工具处理
├── evaluation/             # 评估模块（GAIA、Agentic-RAG）
├── docs/                   # 文档
├── assets/                 # 资源文件
└── setup.cfg               # 项目配置
```

### 核心模块

1. **CLI 入口** (`auto main` / `auto deep-research`)
2. **Agent 编辑器**：自然语言 → Agent 配置
3. **工作流编辑器**：自然语言 → 工作流配置
4. **工具系统**：支持 RapidAPI 等第三方工具平台
5. **评估框架**：GAIA Benchmark、Agentic-RAG

### 技术栈

- Python 3.10+
- Docker（容器化 Agent 交互环境）
- LiteLLM（多模型支持）
- 多 LLM Provider：OpenAI、DeepSeek、Anthropic、Gemini、HuggingFace、Groq 等
