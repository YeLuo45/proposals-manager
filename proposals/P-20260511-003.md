## 项目目标

基于 BMAD/BMAD-METHOD 开源框架，创建设计文档项目，对 agent-assisted 软件交付框架进行系统性分析。

## BMAD-METHOD 概述

BMAD-METHOD 是一个**开源的结构化 Agent 辅助软件交付框架**。

### 核心特性

- **结构化软件交付**：通过 Agent 辅助提升软件交付效率
- **Skill 验证机制**：内置 Skill 验证规则，确保 Agent 行为规范
- **质量门禁**：推送前必须通过 `npm ci && npm run quality` 检查
- **Conventional Commits**：所有提交必须使用 Conventional Commits 格式

### 开发规范

1. **提交规范**：使用 Conventional Commits
2. **质量检查**：推送前运行 `npm ci && npm run quality`
3. **Skill 验证**：规则定义在 `tools/skill-validator.md`
4. **自动化检查**：`npm run validate:skills`（包含在 quality 中）

### 目录结构

```
BMAD-METHOD/
├── AGENTS.md              # 开发规范
├── docs/                  # 文档
├── evals/                 # 评估模块
├── src/                   # 源代码
├── test/                  # 测试
├── tools/                 # 工具（含 skill-validator.md）
├── website/               # 网站
├── eslint.config.mjs     # ESLint 配置
├── prettier.config.mjs    # Prettier 配置

```
